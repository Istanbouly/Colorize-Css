#Introduction

The colorize-basics-linear.css file allows you create basic linear gradients , In this file there is 14 different colors there is 2 basic colors (black & white) and 12 extra colors (red , green , blue , yellow , orange , purple , pink , brown , turquoise , gray , lightblue , lightgreen)

Why black & white are basic colors while the others are extras simply because you can only mix white with any color or mix black with any color but for example you can't mix green and red that's why.

#How to create a linear gradient ?

To create a linear gradient you should use a class that follows this convention grl-degree-color1-color2

grl : stands for gradient linear
degree : here you specify the degree of your gradient
color1 :similar to radial gradient here you specify the first color in the gradient mix
color2 :here you specify the second color in the gradient mix

grl-90-black-white = grl-90-white-black

#Angles Range

The angles (degrees) supported by the library ranges from 10 to 360 in increments of 10 there is also angle 1

#Tips :

  1. We suggest that you use these gradients on small elements like buttons
  2. If you want to use them on larger elements like body for example we suggest you use a degree like 90 or 270 so the gradient never repeat
  3. Take a Couple of seconds and make some custome gradients

#Advanced Linear Gradients

Advanced Linear gradients are listed inside of colorize-specials.css file these are custom gradients they follow the same convention as the basic linear gradients but instead of the color1 & color2 part we replace it with the name of the custom gradient

Here is a table with all of the custom gradients :

peach hazel sel harsheys
royal diana tranqil grapefruit-sunset
solid-vault instagram shrimpy calm-darya
bourbon titanium opa moss
juicy-orange purple-paradise monte-carlo green-beach

Currently the colorize-specials file supports 20 different custom gradients any custom gradient name that consist from two parts, the name parts could be swaped as if the were colors so you don't have to memorize the order of them for example the custom gradient called grapefruit-sunset could be typed as any of these

grl-270-grapefruit-sunset
grl-270-sunset-grapefruit

Notice that these custom gradients are inspired from https://uigradients.com/
