#introduction

The Solids.css file is used to create solid backgrounds or changing the text color it supports 11 different colors each color has 14 different tones (degrees) except black has 8 tones (because black has no darker tones) and white also has 8 (because white has no lighter tones) so in total you have 142 color to chose from don't worry you don't have to memorize the 142 class names we made it easy for you.

#How To Use These Colors ?

First find the element you want to give it a background lets say you have a div element with 100x100px size, and you want to give it a red background then, you should use a class that follows this convention "cl-bg-colorName-tone"

cl : is a namespace for colorize so you don't have conflicts if you are using other similar libraries.

bg : obviously bg stands for background so it tells Colorize Library that you want to use this color for a background not a text color (we are going to learn how to color texts).

colorName : here you replace this part with the color name in our example we want to use red so we are going to type in red our class at this moment should be something like this cl-bg-red this is a correct class and it will use the orginal red color.

Tone :Now for the last part of the class which is the tone here you specify the tone of your selected color the tones ranges from [-6,6], positive numbers means you want a lighter tone so 1 is the minimum lighter tone and 6 is the maximum lighter tone ,on the other hand, negative numbers means a darker tone so -1 is the minimum darker tone and -6 is the maximum darker tone.

lightest tone : cl-bg-red-6
darkest tone : cl-bg-red--6

if you want to use the flat color just replace the tone part with the "flat" flag so it becomes like this

cl-bg-red-flat

#How To Use The Colors For Text ?

it's really simple all you have to do is to replace the bg with fg for (foreground)

Examples :

cl-fg-red
cl-fg-red-6
cl-fg-red--6
cl-fg-red-flat
