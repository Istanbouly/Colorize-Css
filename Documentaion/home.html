Welcome to Colorize.CSS

What is this library (Colorize.css) for ?


This library is used for generating background colors , background linear gradients , background radial gradients and foreground colors really fast just by assigning a class name to the element so it's wonderfull for prototyping.

Structure of this library

There are 4 different basic files :

  1. colorize-basics-linear.css
  2. colorize-basics-radial.css
  3. colorize-specials.css
  4. colorize-solids.css

As the names implies the colorize-basics-linears.css will allow you to create lots of linear gradients with differnt angles just by using the correct class name. The colorize-basics-radial.css will allow you to create radial gradients now for colorize-specials.css This will also allow you to create linear gradients but special ones ( pretty ones ) finally for the colorize-solids it will allow you to create solid backgrounds of different colors with different tones as well, we are going to explain each one in detail.

There are four other files :

  1. colorize-basics-linear.min.css
  2. colorize-basics-radial.min.css
  3. colorize-specials.min.css
  4. colorize-solids.min.css

Obviously these are the minified versions of the original ones and they are the one you are going to use
